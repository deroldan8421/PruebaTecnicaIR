Inscribir Materias


1. Campo "Identificación": Ingrese la cédula del estudiante o el número de identificación en este campo. Al cambiar el valor, se cargará automáticamente el nombre del estudiante en el campo "Nombres del Estudiante".

2. Campo "Profesor": Seleccione el nombre del profesor en el campo correspondiente. Al elegir el profesor, se cargarán las materias asignadas a él o ella.

3. Campo "Materia": Después de seleccionar el profesor, aparecerán las materias asignadas a dicho profesor. Elija la materia que desea inscribir para el estudiante.

Ver Clases

1. Identificación: Ingresa el número de identificación o cédula del estudiante. Al ingresar este dato, el sistema cargará automáticamente las materias a las cuales el estudiante está inscrito.

2. Materias Asignadas: Al llenar el campo de identificación, el campo de "Materias" se llenará con las materias en las que el estudiante está inscrito. Esto permite ver qué materias ya tiene asignadas.

3. Búsqueda de Materias y Estudiantes: Una vez que se seleccione una materia, el sistema mostrará dos listas:

  • Materias del Estudiante: Esta lista mostrará todas las materias en las que el estudiante está inscrito.
  • Estudiantes Inscritos a la Materia: Esta lista mostrará todos los estudiantes que están inscritos en la materia seleccionada, permitiendo ver la información de los estudiantes para cada materia específica.
